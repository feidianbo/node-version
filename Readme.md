# Node-version

A quick module that returns current node version parsed into parts.

## Installation

```shell
  npm install node-version
```

## Quick Start

```js
var currentVersion = require('node-version');

/*
console.log(currentVersion);

{
    original: 'v0.4.10', // same as process.version
    short: '0.4',
    long: '0.4.10',
    major: '0',
    minor: '4',
    build: '10'
}
*/
```
